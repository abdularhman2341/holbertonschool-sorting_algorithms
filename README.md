Sorting algorithms & Big O
Description
Sorting is the process of arranging the elements of a collection in a specific order, most commonly ascending or descending. It is one of the most fundamental problems in computer science, used everywhere from organizing search results and database records to optimizing other algorithms that depend on ordered input.
Different sorting algorithms approach this problem in very different ways — some compare and swap adjacent elements, others divide the data and conquer recursively, and others distribute values into buckets without direct comparison. Each approach comes with its own trade-offs in speed, memory usage, and stability.
This project implements several classic sorting algorithms from scratch in C. Alongside each implementation, it analyzes the algorithm's time complexity in the best, average, and worst cases using Big O notation.
